一 · 计算基础

BLAS、LAPACK、浮点算术、并行计算 — 每年必考的概念简答题

Q1(a) — 高频题
2013–2017
What is BLAS? Give a short explanation of the different BLAS Levels 1, 2 and 3.
什么是 BLAS？简要解释 BLAS Level 1、2、3 的区别。

解答

BLAS（Basic Linear Algebra Subprograms，基本线性代数子程序）是一组标准化的低级线性代数运算接口。它将运算按计算复杂度分为三级：

Level 1 — 向量-向量运算

例如：$y \leftarrow \alpha x + y$（AXPY）、内积 $x^T y$、向量范数 $\|x\|_2$。

运算量：$O(n)$，数据量：$O(n)$，计算/数据比 ≈ $O(1)$。

Level 2 — 矩阵-向量运算

例如：$y \leftarrow \alpha A x + \beta y$（GEMV）。

运算量：$O(n^2)$，数据量：$O(n^2)$，计算/数据比 ≈ $O(1)$。

Level 3 — 矩阵-矩阵运算

例如：$C \leftarrow \alpha A B + \beta C$（GEMM）。

运算量：$O(n^3)$，数据量：$O(n^2)$，计算/数据比 ≈ $O(n)$。

Q1(b) — 高频题
2013–2017
Explain why BLAS Level 3 matrix-matrix product can come close to the peak performance of a processor.
解释为什么 BLAS Level 3 矩阵-矩阵乘法能接近处理器的峰值性能。

解答

关键在于计算量与数据访问量的比值（arithmetic intensity）。

对于 $n \times n$ 矩阵乘法：需 $O(n^3)$ 次浮点运算，但仅需读取 $O(n^2)$ 个数据。这意味着每个从内存加载到缓存的数据元素可以被重复使用 $O(n)$ 次。

通过分块（blocking）技术，将子矩阵放入高速缓存（cache）中，在缓存内完成大量计算后再换入新数据。这使得处理器的浮点运算单元能被充分利用，内存带宽不再是瓶颈，从而接近峰值 FLOPS。

缓存局部性原理

现代 CPU 有多级缓存（L1/L2/L3），访问速度远快于主内存。Level 3 BLAS 通过分块使数据在缓存中被多次重用，避免了反复访问慢速主内存。相比之下，Level 1 和 Level 2 的计算/数据比为 $O(1)$，性能受限于内存带宽（memory-bound）。

Q1(c) — 高频题
2013–2017
What is LAPACK? Explain the relationship to BLAS.
什么是 LAPACK？说明它与 BLAS 的关系。

解答

LAPACK（Linear Algebra PACKage）是建立在 BLAS 之上的高级线性代数库，提供求解线性方程组、最小二乘问题、特征值问题、奇异值分解等功能。

LAPACK 的算法被设计为尽可能调用 BLAS Level 3 的矩阵-矩阵运算（如分块 LU 分解），从而在不同硬件上自动获得接近峰值的性能。用户只需提供高效的 BLAS 实现（如 Intel MKL、OpenBLAS），LAPACK 就能高效运行。

Q1(d)
2013/2015
Explain why vectorized operations in scripting languages are preferable to for-loops.
解释为什么脚本语言中向量化操作优于 for 循环。

解答

像 Python（NumPy）/ MATLAB 这样的脚本语言是解释执行的，每次循环迭代都有巨大的解释器开销（类型检查、动态分派等）。

向量化操作（如 A @ B、np.dot）将整个运算交给底层的编译后的 C/Fortran BLAS 库执行，循环在编译语言层面进行，避免了解释器开销，并能利用 SIMD 指令和缓存优化。

Python 示例 
import numpy as np
import time

n = 10000
a = np.random.rand(n)
b = np.random.rand(n)

# ❌ 慢：Python for 循环
t0 = time.time()
s = 0.0
for i in range(n):
    s += a[i] * b[i]
print(f"for-loop: {time.time()-t0:.4f}s")

# ✅ 快：向量化操作
t0 = time.time()
s = np.dot(a, b)
print(f"vectorized: {time.time()-t0:.6f}s")
Q1(e)
综合
Explain shared memory parallelism vs distributed memory parallelism.
解释共享内存并行与分布式内存并行。

共享内存并行（Shared Memory）

多个线程/处理器核心共享同一块内存地址空间。典型实现为 OpenMP。优点是编程简单（通过指令注解），缺点是可扩展性受限于单机核心数，需注意竞态条件（race condition）。

分布式内存并行（Distributed Memory）

每个处理器有独立的内存空间，通过网络消息传递（message passing）交换数据。典型实现为 MPI。优点是可扩展到数千节点，缺点是编程复杂，需显式管理通信。

Q1(d/e) — 浮点数
2014/2015/2017
Define the machine precision $\epsilon_{\text{mach}}$ and explain what it means. State the fundamental axiom of floating point arithmetic.
定义机器精度 $\epsilon_{\text{mach}}$ 并解释其含义。给出浮点算术的基本公理。

机器精度 $\epsilon_{\text{mach}}$

定义：$\epsilon_{\text{mach}}$ 是满足 $\text{fl}(1 + \epsilon) > 1$ 的最小正浮点数，即浮点系统能区分 $1$ 和 $1+\epsilon$ 的最小增量。

对于 IEEE 双精度（64位），$\epsilon_{\text{mach}} \approx 2.2 \times 10^{-16}$（即 $2^{-52}$）。

含义：任何实数 $x$ 被舍入到浮点数 $\text{fl}(x)$ 时，相对误差不超过 $\epsilon_{\text{mach}}$。

浮点算术基本公理

Fundamental Axiom of Floating Point

对于任意实数 $x$（在浮点可表示范围内），存在 $\delta$ 满足 $|\delta| \le \epsilon_{\text{mach}}$，使得： $$\text{fl}(x) = x(1 + \delta)$$ 对于基本算术运算 $\circ \in \{+, -, \times, \div\}$： $$\text{fl}(a \circ b) = (a \circ b)(1 + \delta), \quad |\delta| \le \epsilon_{\text{mach}}$$

Q1 — 误差分析
2013/2015/2017
Define relative backward error, relative forward error and condition number. When is an algorithm backward stable?
定义相对后向误差、相对前向误差和条件数。什么时候算法是后向稳定的？

设 $f: X \to Y$ 是精确映射，$\tilde{f}$ 是算法的近似计算。对于输入 $x$，算法输出 $\tilde{y} = \tilde{f}(x)$。

相对前向误差 (Relative Forward Error)

$$E_{\text{fwd}} = \frac{\|f(x) - \tilde{f}(x)\|}{\|f(x)\|}$$

度量输出结果与真实值之间的差距。

相对后向误差 (Relative Backward Error)

$$E_{\text{bwd}} = \min\left\{\frac{\|\delta x\|}{\|x\|} : f(x + \delta x) = \tilde{f}(x)\right\}$$

寻找最小扰动 $\delta x$，使得精确算法作用在扰动输入上恰好得到算法的实际输出。

条件数 (Condition Number)

$$\kappa = \lim_{\delta \to 0} \sup_{\|\delta x\| \le \delta \|x\|} \frac{\|f(x+\delta x)-f(x)\| / \|f(x)\|}{\|\delta x\| / \|x\|}$$

描述问题本身对输入扰动的敏感程度。

核心关系

$$\text{前向误差} \lesssim \kappa \times \text{后向误差}$$ 条件数是问题固有的，与算法无关。后向误差度量算法质量。

后向稳定性：如果对任意输入 $x$，算法的输出 $\tilde{f}(x) = f(x + \delta x)$，其中 $\|\delta x\|/\|x\| = O(\epsilon_{\text{mach}})$，则算法是后向稳定的。即算法的输出是某个"稍微扰动的输入"的精确解。

Q2(a) — LU 求解
2013/2014
Solve the following system using LU decomposition without pivoting: $$\begin{bmatrix} 1 & 1 & 1 \\ 2 & 4 & 2 \\ -1 & 5 & 4 \end{bmatrix} \begin{bmatrix} x_1 \\ x_2 \\ x_3 \end{bmatrix} = \begin{bmatrix} 3 \\ 8 \\ 8 \end{bmatrix}$$
用不带行交换的 LU 分解求解上述线性方程组。
LU 分解回顾

将 $A$ 分解为下三角 $L$（对角元为1）和上三角 $U$ 的乘积 $A = LU$。然后通过前代（forward substitution）解 $Ly = b$，再通过回代（back substitution）解 $Ux = y$。

Step 1：LU 分解

消去第一列

$\ell_{21} = a_{21}/a_{11} = 2/1 = 2$

$\ell_{31} = a_{31}/a_{11} = -1/1 = -1$

$R_2 \leftarrow R_2 - 2R_1$：$(2,4,2) - 2(1,1,1) = (0, 2, 0)$

$R_3 \leftarrow R_3 - (-1)R_1$：$(-1,5,4) + (1,1,1) = (0, 6, 5)$

消去第二列

$\ell_{32} = 6/2 = 3$

$R_3 \leftarrow R_3 - 3R_2$：$(0,6,5) - 3(0,2,0) = (0, 0, 5)$

因此：

$$L = \begin{bmatrix} 1 & 0 & 0 \\ 2 & 1 & 0 \\ -1 & 3 & 1 \end{bmatrix}, \quad U = \begin{bmatrix} 1 & 1 & 1 \\ 0 & 2 & 0 \\ 0 & 0 & 5 \end{bmatrix}$$

Step 2：前代 $Ly = b$

Forward Substitution

$y_1 = 3$

$y_2 = 8 - 2 \cdot 3 = 2$

$y_3 = 8 - (-1)(3) - 3(2) = 8 + 3 - 6 = 5$

Step 3：回代 $Ux = y$

Back Substitution

$5x_3 = 5 \Rightarrow x_3 = 1$

$2x_2 = 2 \Rightarrow x_2 = 1$

$x_1 + 1 + 1 = 3 \Rightarrow x_1 = 1$

答案：$x = (1, 1, 1)^T$

Python 验证 
import numpy as np
from scipy.linalg import lu

A = np.array([[1,1,1],[2,4,2],[-1,5,4]], dtype='float')
b = np.array([3,8,8], dtype='float')

# LU 分解
P, L, U = lu(A)
print("L =", L)
print("U =", U)

# 求解
x = np.linalg.solve(A, b)
print("x =", x)  # [1. 1. 1.]

Q2(b) — 条件数界
2013/2015
Let $Ax = b$ and $A\tilde{x} = \tilde{b}$. Show that $$\frac{\|x - \tilde{x}\|}{\|x\|} \le \kappa(A)\frac{\|b - \tilde{b}\|}{\|b\|}$$ where $\kappa(A) = \|A\| \cdot \|A^{-1}\|$.
证明解的相对误差受条件数乘以右端项相对扰动的约束。

证明

设 $\delta b = b - \tilde{b}$，则 $\delta x = x - \tilde{x} = A^{-1}b - A^{-1}\tilde{b} = A^{-1}\delta b$。

Step 1：估计 $\|\delta x\|$

$$\|\delta x\| = \|A^{-1}\delta b\| \le \|A^{-1}\| \cdot \|\delta b\|$$

Step 2：估计 $\|x\|$ 的下界

由 $b = Ax$，得 $\|b\| \le \|A\| \cdot \|x\|$，即：

$$\frac{1}{\|x\|} \le \frac{\|A\|}{\|b\|}$$

Step 3：合并

$$\frac{\|\delta x\|}{\|x\|} \le \|A^{-1}\| \cdot \|\delta b\| \cdot \frac{\|A\|}{\|b\|} = \kappa(A) \frac{\|\delta b\|}{\|b\|} \qquad \blacksquare$$

Q2(b/c) — 三对角 LU
2013/2014
Let $A \in \mathbb{R}^{n \times n}$ be tridiagonal. Show that its LU decomposition can be computed in $O(n)$ operations. Also prove the explicit formulas for $L$ and $U$.
设 $A$ 为三对角矩阵（仅主对角线和上下相邻对角线非零），证明 LU 分解只需 $O(n)$ 次运算，并写出 $L$, $U$ 的递推公式。

三对角矩阵形如：

$$A = \begin{bmatrix} \alpha & \beta \\ \gamma & \alpha & \beta \\ & \ddots & \ddots & \ddots \\ & & \gamma & \alpha & \beta \\ & & & \gamma & \alpha \end{bmatrix}$$

证明

LU 分解中，$L$ 仅有主对角线和下一条副对角线非零，$U$ 仅有主对角线和上一条副对角线非零。

递推公式

$U$ 的上副对角线：$u_{i,i+1} = \beta$（直接保留），$i = 1, \ldots, n-1$

$U$ 的对角线：$u_{1,1} = \alpha$，且对 $i \ge 2$： $$u_{i,i} = \alpha - \frac{\beta \gamma}{u_{i-1,i-1}}$$

$L$ 的下副对角线：对 $i \ge 2$： $$\ell_{i,i-1} = \frac{\gamma}{u_{i-1,i-1}}$$

$L$ 的对角线：$\ell_{i,i} = 1$

运算量分析

每步递推只涉及常数次运算（1次除法 + 1次乘法 + 1次减法），共 $n-1$ 步。因此总运算量为 $O(n)$。

验证 $A = LU$

考虑 $(LU)_{i,i}$：

$$\ell_{i,i} \cdot u_{i,i} + \ell_{i,i-1} \cdot u_{i-1,i} = u_{i,i} + \frac{\gamma}{u_{i-1,i-1}} \cdot \beta$$

由递推：$u_{i,i} = \alpha - \frac{\beta\gamma}{u_{i-1,i-1}}$，所以上式 $= \alpha$。✓

Python — 三对角 LU 分解 
import numpy as np

def tridiag_lu(alpha, beta, gamma, n):
    """三对角矩阵 LU 分解，O(n) 复杂度"""
    u_diag = np.zeros(n)
    l_sub = np.zeros(n - 1)
    u_diag[0] = alpha
    for i in range(1, n):
        l_sub[i-1] = gamma / u_diag[i-1]
        u_diag[i] = alpha - beta * gamma / u_diag[i-1]
    return l_sub, u_diag

# 示例
l_sub, u_diag = tridiag_lu(4.0, 1.0, 1.0, 5)
print("U diagonal:", u_diag)
print("L sub-diagonal:", l_sub)
Q2(c) — Cholesky 分解
2017
Under what conditions does the Cholesky decomposition $A = R^T R$ exist? Show that $\|A\|_2 = \|R\|_2^2$. What is the $LDL^T$ decomposition?
Cholesky 分解 $A = R^T R$ 存在的条件是什么？证明 $\|A\|_2 = \|R\|_2^2$。$LDL^T$ 分解是什么？

Cholesky 分解存在条件

Cholesky 分解定理

$A = R^T R$（$R$ 为上三角矩阵）当且仅当 $A$ 是对称正定的（即 $A = A^T$ 且对所有 $x \ne 0$ 有 $x^T A x > 0$）。

反例：矩阵 $A = \begin{bmatrix} 1 & 2 \\ 2 & 1 \end{bmatrix}$ 是对称的但不正定（特征值为 $3$ 和 $-1$），因此 Cholesky 分解不存在。

证明 $\|A\|_2 = \|R\|_2^2$

$\|A\|_2 = \|R^T R\|_2 = \sigma_{\max}(R^T R)$。

而 $R^T R$ 的特征值就是 $R$ 的奇异值的平方。因此 $\sigma_{\max}(R^T R) = \sigma_{\max}(R)^2 = \|R\|_2^2$。$\blacksquare$

$LDL^T$ 分解

对于一般对称非奇异矩阵 $A$（不必正定），可以分解为 $A = LDL^T$，其中 $L$ 是对角为 1 的下三角矩阵，$D$ 是对角矩阵。

由 LU 分解出发：若 $A = \hat{L}U$，由对称性 $A = A^T = U^T \hat{L}^T$，利用唯一性可得 $U = D\hat{L}^T$，其中 $D = \text{diag}(u_{11}, \ldots, u_{nn})$。

三 · 最小二乘问题

正规方程、SVD 求解、QR 分解求解、增广系统、Givens 旋转

Q3(a) — 正规方程
所有年份 核心
Let $A \in \mathbb{R}^{m \times n}$, $m \ge n$, $\text{rank}(A)=n$, $b \in \mathbb{R}^m$. Find $\hat{x}$ minimizing $\|A\hat{x}-b\|_2$. Show that $A^TA\hat{x} = A^Tb$ is necessary and sufficient.
证明 $A^TA\hat{x} = A^Tb$（正规方程）是最小二乘问题 $\min_x \|Ax-b\|_2$ 的充要条件。

证明

定义 $\phi(x) = \|Ax - b\|_2^2 = (Ax-b)^T(Ax-b) = x^TA^TAx - 2b^TAx + b^Tb$。

必要条件（取梯度）

$$\nabla_x \phi = 2A^TAx - 2A^Tb = 0$$

$$\Rightarrow A^TA\hat{x} = A^Tb$$

充分性

对任意 $x \in \mathbb{R}^n$，令 $x = \hat{x} + v$：

$$\|Ax - b\|_2^2 = \|A\hat{x} - b + Av\|_2^2 = \|A\hat{x}-b\|_2^2 + 2(A\hat{x}-b)^TAv + \|Av\|_2^2$$

由正规方程，$A^T(A\hat{x}-b) = 0$，故交叉项 $= 2(A\hat{x}-b)^TAv = 0$。

因此 $\|Ax-b\|_2^2 = \|A\hat{x}-b\|_2^2 + \|Av\|_2^2 \ge \|A\hat{x}-b\|_2^2$。$\blacksquare$

几何解释

正规方程 $A^T(A\hat{x}-b) = 0$ 意味着残差 $r = b - A\hat{x}$ 与 $A$ 的列空间正交。$\hat{x}$ 使 $A\hat{x}$ 成为 $b$ 在 $\text{Col}(A)$ 上的正交投影。

Q3(b) — 增广系统
2013/2014
Show that $\hat{x}$ is also given by the solution of: $$\begin{bmatrix} I & A \\ A^T & 0 \end{bmatrix}\begin{bmatrix} y \\ \hat{x} \end{bmatrix} = \begin{bmatrix} b \\ 0 \end{bmatrix}$$ What is the interpretation of $y$?
证明增广系统也给出最小二乘解。$y$ 的含义是什么？

证明

展开增广系统：

第一行：$y + A\hat{x} = b \Rightarrow y = b - A\hat{x}$

第二行：$A^T y = 0$

将第一行代入第二行：$A^T(b - A\hat{x}) = 0 \Rightarrow A^TA\hat{x} = A^Tb$

这正是正规方程！$\blacksquare$

$y$ 的解释

$y = b - A\hat{x}$ 就是残差向量（residual），即 $b$ 与其在 $A$ 列空间上投影的差。$A^Ty = 0$ 表明残差与列空间正交。

Q3(c) — SVD 求解
2013 Resit/Mock
Let $A = U\Sigma V^T$ be the SVD. Show that $\hat{x} = V(\Sigma^T\Sigma)^{-1}\Sigma^T U^T b$ and $\|\hat{x}\|_2 \le \sigma_n^{-1}\|b\|_2$.
利用 SVD 求解最小二乘并给出解的范数估计。

SVD 求解

将 $A = U\Sigma V^T$ 代入正规方程 $A^TA\hat{x} = A^Tb$：

$$V\Sigma^T U^T U \Sigma V^T \hat{x} = V\Sigma^T U^T b$$

由 $U$ 正交，$U^TU = I$：

$$V\Sigma^T\Sigma V^T \hat{x} = V\Sigma^T U^T b$$

左乘 $V^T$（$V$ 正交）：

$$\Sigma^T\Sigma V^T\hat{x} = \Sigma^T U^T b$$

因此：$\hat{x} = V(\Sigma^T\Sigma)^{-1}\Sigma^T U^T b$

伪逆 (Moore-Penrose Pseudoinverse)

$A^\dagger = V\Sigma^\dagger U^T$，其中 $\Sigma^\dagger$ 是将 $\Sigma$ 中非零奇异值取倒数、转置得到的矩阵。$\hat{x} = A^\dagger b$。

范数估计

$$\|\hat{x}\|_2 = \|V(\Sigma^T\Sigma)^{-1}\Sigma^T U^T b\|_2$$

$V$ 正交保范数，$(\Sigma^T\Sigma)^{-1}\Sigma^T$ 的最大奇异值为 $\sigma_n^{-1}$（$\sigma_n$ 是 $A$ 的最小奇异值），$U^T$ 保范数，因此：

$$\|\hat{x}\|_2 \le \sigma_n^{-1}\|b\|_2 \qquad \blacksquare$$

Q3(c) — QR 求解
2013 Mock/2014/2017
Use the normal equation to show that $\hat{x} = R^{-1}Q^Tb$ where $A = QR$ is the reduced QR factorization.
利用正规方程证明：通过 QR 分解可得 $\hat{x} = R^{-1}Q^Tb$。

证明

设 $A = QR$ 为约化 QR 分解，$Q \in \mathbb{R}^{m \times n}$ 列正交，$R \in \mathbb{R}^{n \times n}$ 上三角。

代入正规方程 $A^TA\hat{x} = A^Tb$：

$$R^TQ^TQR\hat{x} = R^TQ^Tb$$

$Q$ 列正交故 $Q^TQ = I_n$：

$$R^TR\hat{x} = R^TQ^Tb$$

$R$ 非奇异（因 $\text{rank}(A) = n$），$R^T$ 也非奇异，两边左乘 $(R^T)^{-1}$：

$$R\hat{x} = Q^Tb \quad \Rightarrow \quad \hat{x} = R^{-1}Q^Tb \qquad \blacksquare$$

Python — QR 最小二乘 
import numpy as np

# 超定系统 Ax ≈ b
A = np.array([[1, 1], [1, 2], [1, 3]], dtype='float')
b = np.array([1, 2.1, 2.9])

# 方法1: QR 分解
Q, R = np.linalg.qr(A, mode='reduced')
x_qr = np.linalg.solve(R, Q.T @ b)

# 方法2: lstsq
x_ls, res, _, _ = np.linalg.lstsq(A, b, rcond=None)

Q4(a) — 幂迭代
所有年份 核心
Write down the power iteration to compute the largest magnitude eigenvalue of $A \in \mathbb{R}^{n \times n}$.
写出求矩阵最大模特征值的幂迭代算法。

幂迭代算法

Algorithm: Power Iteration

输入：矩阵 $A \in \mathbb{R}^{n \times n}$，初始向量 $v^{(0)}$（$\|v^{(0)}\|=1$）

对 $k = 1, 2, \ldots$：

 1. $w^{(k)} = Av^{(k-1)}$

 2. $v^{(k)} = w^{(k)} / \|w^{(k)}\|$

 3. $\lambda^{(k)} = (v^{(k)})^T A v^{(k)}$（Rayleigh 商）

输出：$\lambda^{(k)} \to \lambda_1$（最大模特征值），$v^{(k)} \to v_1$（对应特征向量）

收敛性

设 $|\lambda_1| > |\lambda_2| \ge \cdots \ge |\lambda_n|$，则收敛速度为 $O(|\lambda_2/\lambda_1|^k)$。要求 $v^{(0)}$ 在 $\lambda_1$ 对应特征向量方向上有非零分量。

Python — 幂迭代 
import numpy as np

def power_iteration(A, num_iter=100, tol=1e-10):
    n = A.shape[0]
    v = np.random.randn(n)
    v = v / np.linalg.norm(v)
    for _ in range(num_iter):
        w = A @ v
        v_new = w / np.linalg.norm(w)
        lam = v_new @ A @ v_new
        if np.linalg.norm(v_new - v) < tol:
            break
        v = v_new
    return lam, v

A = np.array([[2, 1], [1, 3]], dtype='float')
lam, v = power_iteration(A)
print(f"最大特征值: {lam:.6f}")
print(f"NumPy 验证: {np.linalg.eigvals(A).max():.6f}")
Q4(b) — Rayleigh 商迭代
所有年份
Formulate the Rayleigh quotient iteration for a symmetric matrix $A$ and state its order of convergence.
给出对称矩阵的 Rayleigh 商迭代算法及其收敛阶。

Rayleigh 商迭代

Algorithm

输入：对称矩阵 $A$，初始向量 $v^{(0)}$（$\|v^{(0)}\|=1$）

$\sigma^{(0)} = (v^{(0)})^T A v^{(0)}$

对 $k = 1, 2, \ldots$：

 1. 求解 $(A - \sigma^{(k-1)}I)w^{(k)} = v^{(k-1)}$

 2. $v^{(k)} = w^{(k)} / \|w^{(k)}\|$

 3. $\sigma^{(k)} = (v^{(k)})^T A v^{(k)}$（更新 Rayleigh 商）

收敛阶

对称矩阵的 Rayleigh 商迭代具有三次（cubic）收敛速度，即误差满足 $|\sigma^{(k+1)} - \lambda| = O(|\sigma^{(k)} - \lambda|^3)$。这比普通反幂迭代（线性收敛）和带固定位移的反幂迭代（线性收敛）快得多。

Q4(c) — Rayleigh 商性质
2013 Resit/Mock/2014
Let $\lambda$ be a simple eigenvalue of symmetric $A$ with eigenvector $x$, and $\tilde{x} = x + \epsilon$. Show that $\frac{\tilde{x}^H A \tilde{x}}{\tilde{x}^H \tilde{x}} = \lambda + O(\|\epsilon\|^2)$.
证明特征向量是 Rayleigh 商的驻点，并说明 Rayleigh 商在特征向量附近具有二阶精度。

证明

设 $Ax = \lambda x$，$\|x\| = 1$。定义 $r(v) = \frac{v^H A v}{v^H v}$。

Step 1：展开分子

$$\tilde{x}^H A \tilde{x} = (x+\epsilon)^H A (x+\epsilon) = x^HAx + \epsilon^HAx + x^HA\epsilon + \epsilon^HA\epsilon$$

$$= \lambda + \lambda\epsilon^Hx + \lambda x^H\epsilon + \epsilon^HA\epsilon = \lambda + 2\lambda\text{Re}(x^H\epsilon) + O(\|\epsilon\|^2)$$

（利用 $A$ 对称 $\Rightarrow$ $x^HA\epsilon = \lambda x^H\epsilon$ 和 $\epsilon^HAx = \lambda\epsilon^Hx$）

Step 2：展开分母

$$\tilde{x}^H \tilde{x} = 1 + \epsilon^Hx + x^H\epsilon + \|\epsilon\|^2 = 1 + 2\text{Re}(x^H\epsilon) + O(\|\epsilon\|^2)$$

Step 3：相除

$$r(\tilde{x}) = \frac{\lambda(1 + 2\text{Re}(x^H\epsilon)) + O(\|\epsilon\|^2)}{1 + 2\text{Re}(x^H\epsilon) + O(\|\epsilon\|^2)} = \lambda + O(\|\epsilon\|^2) \qquad\blacksquare$$

这说明 Rayleigh 商在特征向量处是驻点，且以二阶精度逼近特征值——即使特征向量有 $O(\epsilon)$ 误差，Rayleigh 商的误差只有 $O(\epsilon^2)$。

Q4(d) — Gershgorin 圆盘定理
2013 Resit/2014/2015
Show that the eigenvalues of $A$ lie in the union of disks $R_i := \{|z - a_{ii}| \le \sum_{j \ne i} |a_{ij}|\}$.
证明 Gershgorin 圆盘定理：所有特征值都在 Gershgorin 圆盘的并集内。
Gershgorin Circle Theorem

矩阵 $A$ 的每一个特征值 $\lambda$ 都位于至少一个圆盘 $R_i = \{z \in \mathbb{C} : |z - a_{ii}| \le \sum_{j=1, j\ne i}^n |a_{ij}|\}$ 内。

证明

设 $Ax = \lambda x$，$x \ne 0$。取 $i$ 为 $|x_i|$ 最大的分量的下标（即 $|x_i| = \|x\|_\infty$）。

考虑 $Ax = \lambda x$ 的第 $i$ 行：

$$\sum_{j=1}^n a_{ij}x_j = \lambda x_i$$

$$a_{ii}x_i + \sum_{j \ne i} a_{ij}x_j = \lambda x_i$$

$$(\lambda - a_{ii})x_i = \sum_{j \ne i} a_{ij}x_j$$

两边取模：

$$|\lambda - a_{ii}| \cdot |x_i| \le \sum_{j \ne i} |a_{ij}| \cdot |x_j| \le \sum_{j \ne i} |a_{ij}| \cdot |x_i|$$

因为 $|x_i| > 0$，两边除以 $|x_i|$：

$$|\lambda - a_{ii}| \le \sum_{j=1, j \ne i}^n |a_{ij}| \qquad\blacksquare$$

推论：严格对角占优矩阵非奇异

若 $|a_{ii}| > \sum_{j \ne i} |a_{ij}|$ 对所有 $i$ 成立，则所有 Gershgorin 圆盘不包含原点，因此 $\lambda = 0$ 不是特征值，矩阵非奇异。

Q4(a/c) — 特征值扰动
2013/2014
Let $\lambda$ be a simple eigenvalue of $A$ and $\lambda + \delta\lambda$ the corresponding eigenvalue of $A + \delta A$. Show to first order: $\delta\lambda = \frac{y^H \delta A\, x}{y^H x}$.
对于简单特征值的一阶扰动，证明 $\delta\lambda = y^H \delta A\, x / y^H x$（$x$：右特征向量，$y$：左特征向量）。

证明

从扰动方程出发：$(A + \delta A)(x + \delta x) = (\lambda + \delta\lambda)(x + \delta x)$

展开（忽略二阶小量 $\delta A \cdot \delta x$ 和 $\delta\lambda \cdot \delta x$）：

$$Ax + A\delta x + \delta A \cdot x = \lambda x + \lambda\delta x + \delta\lambda \cdot x$$

利用 $Ax = \lambda x$ 消去：

$$A\delta x + \delta A \cdot x = \lambda\delta x + \delta\lambda \cdot x$$

两边左乘左特征向量 $y^H$（满足 $y^H A = \lambda y^H$）：

$$y^H A\delta x + y^H\delta A\cdot x = \lambda y^H\delta x + \delta\lambda \cdot y^H x$$

因为 $y^HA = \lambda y^H$，所以 $y^HA\delta x = \lambda y^H\delta x$，对消得：

$$y^H\delta A\cdot x = \delta\lambda \cdot y^Hx$$

$$\boxed{\delta\lambda = \frac{y^H\delta A\cdot x}{y^H x}} \qquad\blacksquare$$

对称矩阵的简化

对称矩阵的左右特征向量相同（$y = x$），因此 $\delta\lambda = \frac{x^H\delta A\, x}{x^Hx} = x^H\delta A\, x$（归一化情形）。这意味着对称矩阵的特征值总是 well-conditioned 的。

Q4(b) — Schur 分解
2017
Prove that every square matrix $A \in \mathbb{C}^{n \times n}$ has a Schur decomposition $A = QTQ^H$ where $Q$ is unitary and $T$ is upper triangular.
证明每个方阵都有 Schur 分解。

归纳法证明

基础：$n = 1$

$A = [\lambda]$ 已经是上三角，$Q = [1]$。✓

归纳步骤

设结论对 $n-1$ 阶矩阵成立。设 $\lambda_1$ 是 $A$ 的特征值，$q_1$ 是归一化特征向量。

将 $q_1$ 扩展为酉基 $\hat{Q} = [q_1 \mid Q_\perp]$，则：

Q5(a) — Lagrange 插值
所有年份 核心
Write down the Lagrange interpolation polynomial of degree $n$ that interpolates $f$ in the points $x_0, \ldots, x_n$.
写出过 $n+1$ 个点的 $n$ 次 Lagrange 插值多项式。

Lagrange 插值公式

$$p(x) = \sum_{k=0}^n f(x_k) L_k(x)$$

其中 Lagrange 基多项式为：

$$L_k(x) = \prod_{\substack{j=0 \\ j \ne k}}^n \frac{x - x_j}{x_k - x_j}$$

关键性质

$L_k(x_i) = \delta_{ki}$（在 $x_k$ 处为 1，在其他节点处为 0）。因此 $p(x_k) = f(x_k)$ 对所有 $k$ 成立。

Q5(b) — 插值唯一性
2013 Mock/2015
Show that there exists exactly one polynomial $p$ of degree $\le N$ such that $p(x_j) = y_j$ for $j = 0, \ldots, N$.
证明满足 $N+1$ 个插值条件的不超过 $N$ 次多项式是唯一的。

存在性

Lagrange 插值公式显式构造了这样一个多项式，因此存在性成立。

唯一性（反证法）

假设 $p$ 和 $q$ 都是不超过 $N$ 次的多项式且在所有节点处相等。

令 $r(x) = p(x) - q(x)$，则 $r$ 的次数 $\le N$，且 $r(x_j) = 0$ 对 $j = 0, \ldots, N$ 成立。

一个不超过 $N$ 次的多项式有 $N+1$ 个零点，而不超过 $N$ 次的非零多项式最多有 $N$ 个零点。矛盾。

因此 $r \equiv 0$，即 $p \equiv q$。$\blacksquare$

Q5(c) — Neville 递推
2013 Resit/Mock/2014
Let $q$ be the interpolation polynomial for $x_0, \ldots, x_N$ and $r$ for $x_1, \ldots, x_{N+1}$. Show: $$p(x) = \frac{(x-x_0)r(x) - (x-x_{N+1})q(x)}{x_{N+1} - x_0}$$ is the interpolation polynomial for all $N+2$ points.
证明 Neville 递推公式：用两个低阶插值多项式构造高阶插值多项式。

证明

需验证 $p$ 的次数和插值条件。

次数检查

$q$ 和 $r$ 的次数均为 $N$。$(x-x_0)r(x)$ 和 $(x-x_{N+1})q(x)$ 的次数为 $N+1$。相减后除以常数，次数 $\le N+1$。✓

在 $x_0$ 处

$$p(x_0) = \frac{0 \cdot r(x_0) - (x_0-x_{N+1})q(x_0)}{x_{N+1}-x_0} = \frac{(x_{N+1}-x_0)y_0}{x_{N+1}-x_0} = y_0 \quad\checkmark$$

在 $x_{N+1}$ 处

$$p(x_{N+1}) = \frac{(x_{N+1}-x_0)r(x_{N+1}) - 0}{x_{N+1}-x_0} = r(x_{N+1}) = y_{N+1} \quad\checkmark$$

在 $x_i$（$1 \le i \le N$）处

$q(x_i) = y_i$，$r(x_i) = y_i$，因此：

$$p(x_i) = \frac{(x_i-x_0)y_i - (x_i-x_{N+1})y_i}{x_{N+1}-x_0} = y_i \cdot \frac{x_{N+1}-x_0}{x_{N+1}-x_0} = y_i \quad\checkmark$$

由唯一性，$p$ 就是过所有 $N+2$ 点的插值多项式。$\blacksquare$

递推算法

这给出了一种递推计算插值多项式值的方法（Neville 算法 / Aitken 算法）：从常数多项式（各点的值）出发，逐步合并相邻点集的插值多项式。

Q5(b) — Runge 现象
2014
Explain the Runge phenomenon and how it can be prevented.
解释 Runge 现象及其预防方法。

Runge 现象

对某些函数（如 $f(x) = 1/(1+25x^2)$），使用等距节点的高次多项式插值时，随着次数增加，插值多项式在区间端点附近产生剧烈振荡，误差不减反增。

预防方法

使用 Chebyshev 节点（集中在端点附近）：

$$x_k = \cos\left(\frac{2k+1}{2(n+1)}\pi\right), \quad k = 0, \ldots, n$$

Chebyshev 节点使得节点多项式 $\omega(x) = \prod_{k}(x-x_k)$ 的最大模最小化，从而控制插值误差。

Q5(b) — 最佳逼近
2013 Resit/Mock
Give the definition of the best approximating polynomial and state the difference between interpolation and approximation.
定义最佳逼近多项式，并说明插值与逼近的区别。

最佳逼近多项式

$n$ 次最佳逼近多项式 $p^*$ 满足：

$$\|f - p^*\|_\infty = \min_{p \in \mathcal{P}_n} \|f - p\|_\infty = \min_{p \in \mathcal{P}_n} \max_{x \in [a,b]} |f(x) - p(x)|$$

插值 vs 逼近

插值：要求多项式在有限个节点处精确等于函数值 $p(x_k) = f(x_k)$，但在其他点处可能有较大误差。

逼近：不要求在任何特定点精确相等，而是在整个区间上最小化某种范数意义下的误差。逼近通常在全局意义上更好。

Q6(a) — 梯形法则
所有年份 核心
Derive the Trapezoid rule for approximating $I(f) = \int_a^b f(x)\,dx$.

推导

用 $a$ 和 $b$ 两点的线性插值来近似 $f$：

$$p_1(x) = f(a) \cdot \frac{b-x}{b-a} + f(b) \cdot \frac{x-a}{b-a}$$

积分：

$$I_1(f) = \int_a^b p_1(x)\,dx = f(a)\int_a^b \frac{b-x}{b-a}dx + f(b)\int_a^b\frac{x-a}{b-a}dx$$

计算权重

$$\int_a^b \frac{b-x}{b-a}dx = \frac{1}{b-a}\left[bx - \frac{x^2}{2}\right]_a^b = \frac{b-a}{2}$$

同理，$\int_a^b \frac{x-a}{b-a}dx = \frac{b-a}{2}$

因此梯形法则为：

$$\boxed{I_1(f) = \frac{b-a}{2}[f(a) + f(b)]}$$

误差估计

$$|I(f) - I_1(f)| \le \frac{(b-a)^3}{12}M_2, \quad M_2 = \max_{x \in [a,b]}|f''(x)|$$

周期函数简化

对于周期函数，$f(0) = f(2\pi)$，首尾半权重合并：

$$\boxed{I_m(f) = h\sum_{k=0}^{m-1} f(x_k) = \frac{2\pi}{m}\sum_{k=0}^{m-1} f(kh)}$$

所有权重相同的直觉解释

对于周期函数，每个节点在区间中的"地位"完全对等——没有端点效应（端点就是内点的周期延拓）。每个点代表宽度为 $h$ 的一小段，用该点的函数值乘以 $h$ 来近似该段面积，所以所有权重都等于 $h$。

Q6(d) — 三角域积分
2013/2017
Let $T$ be the unit triangle with vertices $(0,0), (1,0), (1,1)$. Show that $$\int_T g(x,y)\,dydx \approx \sum_{i,j=0}^n g(\xi_i, \xi_j\xi_i)\,\xi_i\omega_i\omega_j$$ using substitution $y = \mu x$.
利用变量替换将三角形上的积分化为两个 $[0,1]$ 上的一维积分，再用求积公式近似。

推导

原积分：$\int_0^1 \int_0^x g(x,y)\,dy\,dx$

变量替换 $y = \mu x$

固定 $x$，令 $y = \mu x$，$dy = x\,d\mu$。当 $y: 0 \to x$ 时，$\mu: 0 \to 1$。

$$\int_0^x g(x,y)\,dy = \int_0^1 g(x, \mu x)\,x\,d\mu$$

双重积分

$$\int_T g\,dydx = \int_0^1 \int_0^1 g(x, \mu x)\,x\,d\mu\,dx = \int_0^1 x\left[\int_0^1 g(x,\mu x)\,d\mu\right]dx$$

两个积分都在 $[0,1]$ 上！

应用求积公式

对内层（$\mu$ 积分）用 $I_n$：$\int_0^1 g(x, \mu x)d\mu \approx \sum_{j=0}^n g(x, \xi_j x)\omega_j$

对外层（$x$ 积分，含因子 $x$）用 $I_n$：

$$\int_0^1 x \sum_j g(x,\xi_j x)\omega_j\,dx \approx \sum_{i=0}^n \xi_i \sum_{j=0}^n g(\xi_i, \xi_j\xi_i)\omega_j\,\omega_i$$

$$\boxed{\int_T g(x,y)\,dydx \approx \sum_{i,j=0}^n g(\xi_i, \xi_j\xi_i)\,\xi_i\omega_i\omega_j} \qquad\blacksquare$$
